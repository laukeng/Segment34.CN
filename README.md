# Segment34.CN
修改自Segment34 MkII表盘([https://github.com/ludw/Segment34mkII](https://github.com/ludw/Segment34mkII))，感谢原作者开源！添加了农历和二十四节气显示，顺便对设置菜单和天气也做了汉化。由于没有制作点阵风格的汉字字体，当显示农历时请将时钟上方的字体设为“线条”（默认值），以确保能正常显示汉字。
